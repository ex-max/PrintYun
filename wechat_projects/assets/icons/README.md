## 图标规格建议

- 尺寸：推荐 128x128 像素或更大
- 格式：PNG（支持透明背景）
- 颜色：单色图标建议使用深色（#333333），便于通过 CSS 调整颜色

## 图标来源推荐

1. [阿里巴巴矢量图标库](https://www.iconfont.cn/)
2. [Flaticon](https://www.flaticon.com/)
3. [IconPark](https://iconpark.oceanengine.com/)

## 注意事项

- 所有图标文件名必须与代码中引用的名称一致
- 建议使用统一的图标风格
- 图标文件大小建议控制在 50KB 以内
