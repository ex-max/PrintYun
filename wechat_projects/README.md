# PrintYun 微信小程序开发说明

## 项目概述

PrintYun 微信小程序是 PrintYun 云打印平台的移动端应用，为用户提供便捷的手机下单、到店取件的打印服务。

## 技术栈

- **前端框架**：微信小程序原生框架
- **UI 组件**：WeUI 样式库
- **开发工具**：微信开发者工具

## 待开发功能

### 后端 API 接口
需要后端开发以下接口（前缀 `/api/mp/`）：

#### 认证模块
- `POST /auth/login` - 微信登录
- `POST /auth/refresh` - 刷新 Token
- `POST /auth/logout` - 退出登录
- `POST /auth/bindPhone` - 绑定手机号

#### 文件上传模块
- `POST /upload` - 上传文件
- `GET /upload/convertStatus` - 查询文件转换状态

#### 订单模块
- `GET /orders` - 获取订单列表
- `GET /orders/:tradeNumber` - 获取订单详情
- `POST /orders` - 创建订单
- `POST /orders/:tradeNumber/cancel` - 取消订单
- `POST /orders/:tradeNumber/reprint` - 再次打印
- `GET /orders/:tradeNumber/status` - 查询订单状态
- `POST /orders/idcard` - 创建身份证打印订单

#### 支付模块
- `POST /pay/wechat` - 获取微信支付参数

#### 用户模块
- `GET /user/info` - 获取用户信息

#### 打印点模块
- `GET /printPlaces` - 获取打印点列表

#### 打印参数模块
- `GET /printOptions` - 获取打印参数选项

### 小程序配置
需要在微信小程序后台完成以下配置：

1. **服务器域名配置**
   - request 合法域名：后端 API 域名
   - uploadFile 合法域名：文件上传域名
   - downloadFile 合法域名：文件下载域名

2. **业务域名配置**
   - WebView 业务域名（如需内嵌 H5 页面）

3. **隐私协议配置**
   - 用户信息收集说明
   - 位置信息使用说明
   - 相机/相册使用说明

4. **订阅消息配置**
   - 订单状态通知模板
   - 打印完成通知模板

## 开发建议

### 后端对接
1. 修改 `utils/api.js` 中的接口地址
2. 在 `app.js` 中配置 `baseUrl`
3. 根据实际接口返回格式调整数据处理逻辑

### 样式定制
1. 修改 `app.wxss` 中的 CSS 变量调整主题色
2. 图标资源放置到 `assets/icons/` 目录
3. 根据设计稿调整各页面样式

### 功能扩展
1. 添加消息订阅功能
2. 添加优惠券功能
3. 添加会员等级功能
4. 添加打印预览功能
5. 添加文件管理功能

## 注意事项

1. **文件上传**：微信小程序单次上传文件大小限制为 50MB
2. **支付功能**：需要开通微信支付商户号
3. **定位功能**：需要在 `app.json` 中配置 `permission` 字段
4. **登录流程**：需要后端实现微信登录接口（code2Session）
5. **图片资源**：所有图标需要自行准备并放置到 `assets/icons/` 目录
