用户指南

几步即可完成云打印，文件自动排队，支付后到店取件

云打印是一款面向打印店（尤其是学校打印店）和个人用户的在线打印服务平台。用户在线上传文件、在线选择参数、在线支付，到店凭订单号取件，打印机会根据订单信息自动打印，解决排队拥堵和 U 盘传输繁琐的痛点。

一、如何注册

  1. 点击右上角 注册 按钮，进入注册页面；
  2. 输入 11 位手机号，点击 获取验证码，等待短信到达；
  3. 输入短信验证码、设置密码（两次一致），提交即可完成注册；
  4. 注册成功后，系统会跳转到登录页面，使用手机号 + 密码登录即可。

二、如何打印

  1. 登录后点击右上角 开始打印，进入上传页面；
  2. 选择打印点（门店）、上传文件（支持 doc/docx/xls/xlsx/ppt/pptx/pdf/jpg/png）；
  3. 选择打印参数：黑白/彩色、单面/双面、横向/纵向、份数、打印日期；
  4. 点击提交，系统自动将 Office 文档转换为 PDF 并计算页数、单价；
  5. 确认无误后进入支付页面。

三、如何支付

  • 当前支持 微信支付 和 支付宝（通过 XorPay 聚合）；
  • 提交订单后页面自动生成二维码，使用对应 App 扫码支付即可；
  • 支付成功后系统自动更新订单状态为"已支付"，订单进入打印队列。

四、如何取件

  1. 支付成功后，订单自动进入门店打印队列；
  2. 到门店后向店员报出订单号（日期-序号 格式）；
  3. 核对订单信息无误后，打印机将自动开始打印。

五、如何查询订单

  • 登录后进入 订单查询 页面；
  • 可按手机号 + 日期组合查询；
  • 订单状态分为：未支付 / 已支付 / 正在打印 / 打印完成。

六、如何找回密码

  1. 在登录页点击 忘记密码；
  2. 输入注册时的手机号，获取短信验证码；
  3. 输入验证码和新密码，提交即可重置密码；
  4. 如遗忘手机号或无法接收验证码，可拨打客服电话 535027650 进行人工找回。

七、常见问题

Q：支持哪些文件格式？ doc/docx/xls/xlsx/ppt/pptx/pdf/jpg/jpeg/png，其他格式请先转换。
Q：文件上传后多久能打印？ 支付完成后订单自动入队，一般 1-3 分钟可完成打印。
Q：订单错了怎么办？ 未支付订单可直接在"我的订单"删除；已支付订单请联系客服处理。
Q：如何打印彩色？ 在参数选择页将颜色改为"彩色"，单价：单面 1.0 元/页、双面 1.7 元/页。黑白单面 0.3、双面 0.5 元/页。
Q：文件会被保存多久？ 打印完成后会从打印队列移除；长期不访问的文件系统会定期清理。
Q：遇到打印异常如何处理？ 卡纸、断电等导致异常，请拨打客服 535027650 或在公众号反馈。
